# Report Generation Targets

## Output Format

- **Document type**: Scientific/technical report (thesis style), ~52 pages
- **Citation style**: IEEE (numeric `[1]`) via `scripts/writeup/citation-styles/ieee.csl`
- **PDF engine**: XeLaTeX (pandoc `--pdf-engine=xelatex`)
- **Assembly**: pandoc over `results/sections/00_frontmatter.md` +
  `00_title_abstract.tex` (LaTeX title page, passed `-B`) + numbered
  `01_*.md … 08_*.md` + `references.bib`; List of Figures / List of Tables are
  auto-generated by pandoc.

## Style References

### Citation Format

- IEEE numeric style. Bibliography lives in `results/sections/references.bib`
  (BibTeX). Each entry: title, author(s), year, link, accessed date.

### Figure and Table Examples

- Figures: PlantUML sources under `results/assets/diagrams/`, rendered to PNG via
  `scripts/writeup/generate-plantuml-png.sh`, referenced with academic captions.
- Tables: comparison/metric tables in Markdown; pandoc renders them into the LoT.
